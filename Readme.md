# 🎣 SmartFishing - 智能摸鱼工具

![Version](https://img.shields.io/badge/version-2.0.0.0-blue)
![Language](https://img.shields.io/badge/language-aardio-orange)
![Platform](https://img.shields.io/badge/platform-Windows-green)

一款智能摸鱼小工具，用于模拟办公状态，防止远程监控软件检测到异常活动。

## ✨ 功能特性

- 🖱️ **智能鼠标模拟**：随机进行鼠标点击、滚轮滚动等操作
- ⌨️ **键盘防锁屏**：定期按下NumLock或Insert键防止系统锁屏
- 🎛️ **高度可定制**：可调节各种参数，包括点击频率、位置偏移等
- 📌 **窗口管理**：支持置顶、贴边、留边等窗口显示选项
- 🔄 **托盘支持**：最小化到系统托盘，支持右键菜单操作
- 💾 **配置保存**：自动保存用户设置，下次启动时恢复

## 📋 系统要求

- Windows 操作系统
- 无需额外依赖，使用aardio开发

## 🚀 安装使用

### 下载安装

1. 从 [Releases](https://github.com/yourusername/SmartFishing/releases) 下载最新版本
2. 解压到任意目录
3. 运行主程序即可

### 使用方法

1. 打开一个Word文档或其他办公软件
2. 运行SmartFishing程序
3. 根据需要调整各项参数：
   - **鼠标点击**：设置左键/右键点击的频率和位置范围
   - **滚轮动作**：设置上下滚动的频率和幅度
   - **模拟按键**：选择NumLock或Insert键来防止锁屏
   - **时间控制**：调整各种操作的延时时间
4. 点击"Go"按钮开始模拟办公
5. 点击"Stop"按钮停止模拟

## 🎛️ 参数说明

| 功能区域 | 参数 | 说明 |
|---------|------|------|
| **鼠标点击** | 左键/右键 | 启用左键或右键点击 |
| | X/Y坐标 | 点击位置基准坐标 |
| | 随机范围 | 点击位置的随机偏移范围 |
| **滚轮动作** | 上滚/下滚 | 启用向上或向下滚动 |
| | 滚动时间 | 单次滚动持续时间 |
| | 随机范围 | 滚动时间的随机偏移 |
| **模拟按键** | NumLock/Insert | 选择要模拟的按键 |
| | 按键间隔 | 按键操作的时间间隔 |
| **时间控制** | 小延时 | 操作之间的基础延时 |
| | 大延时 | 大循环之间的延时 |
| | 随机范围 | 延时时间的随机偏移 |
